# Real-Time Analytics Game 🎮📊

A backend web service for tracking real-time game analytics using WebSocket, Express.js, and MongoDB.

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- WebSocket (Socket.IO)

## 🚀 Features
- Realtime analytics updates
- WebSocket broadcasting
- Modular API structure

## 📦 Setup
```bash
npm install
npm start
